# FastAPI Learning Project

This project is a simple yet complete FastAPI application built as a learning exercise. It demonstrates how to create a RESTful API with user authentication, CRUD (Create, Read, Update, Delete) operations, and a persistent database using SQLite.

## Features

-   **User Authentication:**
    -   User registration with password hashing (`passlib`).
    -   User login using OAuth2 with Bearer Tokens (JWT).
    -   Protected endpoints that require authentication.
-   **Todo Management (CRUD):**
    -   Create new todo items.
    -   Read all todo items for the authenticated user.
    -   Read a single todo item by ID.
    -   Update an existing todo item.
    -   Delete a todo item.
-   **Database Integration:**
    -   Uses `SQLAlchemy` for ORM and schema definition.
    -   Connects to a `SQLite` database for data persistence.
    -   Asynchronous database operations using the `databases` library.
-   **Interactive API Documentation:**
    -   Automatic interactive API documentation generated by FastAPI (via Swagger UI) at `/docs`.

## Technologies Used

-   **Backend:**
    -   **FastAPI:** The web framework.
    -   **Uvicorn:** The ASGI server to run the application.
-   **Database:**
    -   **SQLAlchemy:** For ORM and schema management.
    -   **SQLite:** The file-based database.
    -   **databases:** For asynchronous database access.
-   **Authentication:**
    -   **passlib[bcrypt]:** For hashing passwords.
    -   **python-jose[jwt]:** For creating and verifying JWT tokens.
-   **Data Validation:**
    -   **Pydantic:** For data validation and serialization.

## Setup and Installation

1.  **Clone the repository (if applicable) or ensure you have the project files.**

2.  **Create a Python virtual environment:**
    ```bash
    python -m venv venv
    ```

3.  **Activate the virtual environment:**
    -   **Windows:**
        ```bash
        .\venv\Scripts\activate
        ```
    -   **macOS/Linux:**
        ```bash
        source venv/bin/activate
        ```

4.  **Install the required dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
    *(Note: You can generate a `requirements.txt` file from the current environment by running `pip freeze > requirements.txt`)*

## Running the Application

Once the setup is complete, you can run the application with the following command:

```bash
uvicorn main:app --reload
```

The server will start, and the application will be accessible at `http://127.0.0.1:8000`.

## How to Test

The easiest way to test the API is through the interactive documentation:

1.  Navigate to `http://127.0.0.1:8000/docs` in your web browser.
2.  **Register a new user** using the `POST /register/` endpoint.
3.  **Log in to get an access token** using the `POST /token` endpoint with your new user's credentials.
4.  **Authorize your session** by clicking the "Authorize" button at the top right and entering your access token (e.g., `Bearer <your_token>`).
5.  You can now test all the protected Todo endpoints.
